Configurer Discord

La configuration de Discord se fait à l'aide des Webhooks des serveurs. Les notifications sont envoyées dans un salon textuel (text channel).

Pour lier votre serveur, il sufit de copier-coller le lien du Webhook dans l'onglet paramètres de Notifius.

Voici un lien vers la documentation de Discord pour la création d'un Webhook : Documentation
